## 2. Detailed Execution Lifecycle

### Phase 1: Syntactic Code Generation
* [cite_start]The Coder Agent ingests the compact Mathematical Brief generated by the state manager[cite: 46, 58].
* [cite_start]Its exclusive responsibility is to evaluate this structured information and write the exact JSON instructions or low-level SymPy code needed to compute the target solution[cite: 58].

### Phase 2: Security Sanitization Guardrails
* [cite_start]Before execution, the raw code string is routed through the deterministic `Sanitizer` script[cite: 63, 64].
* [cite_start]The script leverages Python's native `ast` module or regex patterns to check for structural syntax errors[cite: 63, 64].
* [cite_start]The Sanitizer acts as a system firewall, scanning for malicious commands like `os.system("rm -rf")`[cite: 64].
* [cite_start]If validation issues or safety warnings are tripped, the Sanitizer immediately rejects the code block and stops execution[cite: 65].

### Phase 3: Execution & Localized Healing Loops
* [cite_start]Code payloads that pass sanitization are sent directly to the SymPy Engine for execution[cite: 11, 31].
* [cite_start]The system wraps the execution process in localized compiler-refiner conditional loops using standard `try-except` blocks to handle unexpected runtime faults[cite: 44, 60].
* [cite_start]If a runtime error occurs (such as an undefined variable exception), a Debugger Agent is programmatically triggered[cite: 12, 60, 61].
* [cite_start]The Debugger receives the raw exception trace alongside the faulty script and generates a localized "refined logic" patch[cite: 62].
* [cite_start]This patch is routed back into the validation loop without involving the primary Translator Agent, protecting the master plan from cascading logic failures[cite: 44, 62].

### Phase 4: Result Delivery
* [cite_start]When the SymPy engine completes the computation without exceptions, it outputs a checked and verified result[cite: 13, 38].
* [cite_start]The system evaluates the computed mathematical values and returns the finalized answer cleanly to the user[cite: 14, 39].
